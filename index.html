ÇALIŞMA PRENSİBİHOW IT WORKS

Bir agent görevi nasıl yürütür?How does an agent run a task?

Telefon açmaktan randevu onayına kadar 4 adım — tamamen otomatik.From incoming message to confirmed appointment — 4 steps, fully automated.

1
TetikleyiciTrigger
WhatsApp, e-posta, web formu veya zamanlama. Agent uyarılır ve bağlamı okur.WhatsApp, email, web form, or schedule. Agent wakes up and reads context.
2
AnlamaUnderstand
İsteği kategorize eder, eksik bilgi varsa sorar, aciliyeti belirler.Categorizes the request, asks for missing info if needed, determines urgency.
3
EylemAct
Takvim açar, form doldurur, belge oluşturur, sistem günceller — çok adımlı.Opens calendar, fills form, creates document, updates system — multi-step chain.
4
TeyitConfirm
Müşteriye / ilgili tarafa sonuç iletilir. İstisna varsa insan uyarılır.Notifies customer and relevant parties. Escalates to human only on exceptions.
